# PowerShell
Tristen's PowerShell setup.

Scoop: 
```
iwr -useb get.scoop.sh | iex
```

Oh My Posh: 
```
iwr -useb get.scoop.sh | iex
```
```
scoop update oh-my-posh
```

Terminal-Icons: 
```
Install-Module -Name Terminal-Icons -Repository PSGallery -Force
```

Neovim: 
```
scoop install neovim
```

Git: 
```
winget install --id Git.Git -e --source winget
```
